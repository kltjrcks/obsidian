

0. Inbox
	- 분류되기 전 메모가 모이는곳
	- para 정리법의 분류 방법은 주제별 분류가 아닌 실행별 분류이다.
1. Project
	- 끝이 명확함 (기간 명시)
	- 실행도가 가장 높음
	- ex) 자격증합격하기, 시험공부, 블로그 글 작성, 이사하기 등 
2. Area
	- 끝이 없음, 진행중,
	- 나의 책임이나 역할이 존재
	- ex) 운동, 건강, 경제 계획, 직업관련, 가족관련
3. Resource
	- 내가 관심있는 분야
	- 책임이나 역할이 존재하지 않는 자유로운 주제
	- ex) 글쓰기, 독서, 기타 등
4. Archive
	- 실행도가 0인곳.
	- 종료되거나 중단된 프로젝트
	- 책임이나 관심이 사라진 자원에 대한 창고
